simple ddos code that is capable of attacking unprotected servers


Set up for Termux
```
apt install git
apt install python3
git clone https://github.com/yuhertz/main-ddos
cd main-ddos
python3 ddos.py
```

Set up for Kali Linux
```
sudo apt install git
sudo apt install python3
git clone https://github.com/yuhertz/main-ddos
cd main-ddos
python3 ddos.py
```
